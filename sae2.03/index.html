Présentation générale

Bienvenue sur le site officiel de notre projet Snake Multijoueur en Ligne ! Inspiré du classique jeu "Snake" connu de tous, notre projet permet à deux joueurs de s'affronter simultanément en temps réel, directement depuis leur navigateur web, sans aucune installation.

Objectifs du Projet

  • Expérience de jeu simple et intuitive
  • Connexion rapide via codes de partie
  • Synchronisation efficace des mouvements
  • Serveur robuste pour parties simultanées
  • Déploiement simplifié avec Docker

Côté Client

Interface développée en HTML5, CSS3 et JavaScript avec Socket.io pour une communication temps réel.

  • Connexion via code unique
  • Contrôles intuitifs (flèches ou ZQSD)
  • Affichage temps réel des positions
  • Gestion des collisions côté serveur

Côté Serveur

Développé en Node.js avec Express.js et Socket.io, conteneurisé avec Docker pour une meilleure portabilité.

  • Gestion des salles de jeu
  • Synchronisation parfaite
  • Boucle de jeu optimisée
  • Nettoyage automatique

Comment jouer ?

  1. Créez ou rejoignez une partie avec un code.
  2. Attendez qu'un adversaire se connecte.
  3. Contrôlez votre serpent avec les flèches.
  4. Jouer !!
